MapleKit

A full suite of developer tools, from logging to preferences for use with Maple

MapleKit was designed to use with Maple, and it’s suite of MacOS customization tools. before deciding to build a project with these tools, please make sure you have read everything on Maple’s Website, and that you understand the risks and rewards of using this software.

Docs

For broad guides and how-to’s, please Visit Maple for Developers

For usage details, and other documentation, please visit MapleKit’s Documentation Website

Installation

To use, add MapleKit as a Package Dependency to your Xcode Command Line Project
